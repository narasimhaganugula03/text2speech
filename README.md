# Text To Speech Converter

A Text-to-Speech (TTS) web application that converts written text into natural-sounding speech. Users can enter text and listen to the generated voice output directly in the browser.

## 🚀 Features

- Convert text into speech
- Play voice output instantly
- Simple and clean user interface
- Supports different voices
- Adjustable speech speed
- Responsive design
- Easy-to-use text input

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### APIs
- Web Speech API (SpeechSynthesis)
